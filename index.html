Política de Privacidad

Última actualización: mayo 2025

1. Recolección de Datos

Se recopila el número de teléfono y nombre de usuario únicamente para gestionar consultas vía WhatsApp.

2. Uso de la Información

Los datos se utilizan exclusivamente para el funcionamiento del chatbot universitario.

3. Almacenamiento

Los datos se almacenan de forma segura y no se comparten con terceros.

4. Eliminación de Datos

El usuario puede solicitar la eliminación de sus datos enviando un mensaje al administrador del sistema.

5. Contacto

Para consultas sobre privacidad contactar al administrador institucional.
